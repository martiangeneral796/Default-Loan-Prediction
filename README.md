# Loan Default Prediction Model

This project uses Python and TensorFlow/Keras to predict loan default risk.

Features:
- Data preprocessing
- Artificial Neural Network (ANN)
- Model training and evaluation

Tools:
- Python
- NumPy
- TensorFlow/Keras

Data Features
- Monthly Usage
- Complaints
- Subscription Length
- Monthly Spend

Objective
Predict whether a customer is likely to default on a loan
